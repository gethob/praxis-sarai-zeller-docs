# Design System

The Praxis Sarai Zeller website uses a comprehensive design system built on CSS custom properties (variables) for consistency and maintainability.

## Color Palette

The color scheme reflects a warm, therapeutic, and professional atmosphere.

### Primary Colors

```css
:root {
  --color-primary: #D94A38;      /* Red-orange - Main brand color */
  --color-secondary: #FFED4E;    /* Yellow - Accent color */
  --color-accent: #F5A623;       /* Orange - Highlights */
}
```

#### Color Usage

- **Primary (`#D94A38`)**: CTAs, links, important headings, navigation highlights
- **Secondary (`#FFED4E`)**: Backgrounds, decorative elements, subtle highlights
- **Accent (`#F5A623`)**: Hover states, focus indicators, special emphasis

### Neutral Colors

```css
:root {
  --color-text: #2D3748;         /* Dark gray for body text */
  --color-text-light: #718096;   /* Light gray for secondary text */
  --color-bg: #FFFFFF;           /* White background */
  --color-bg-alt: #F7FAFC;       /* Off-white for alternating sections */
  --color-border: #E2E8F0;       /* Light gray for borders */
}
```

### Semantic Colors

```css
:root {
  --color-success: #48BB78;      /* Green for success states */
  --color-warning: #ED8936;      /* Orange for warnings */
  --color-error: #F56565;        /* Red for errors */
  --color-info: #4299E1;         /* Blue for information */
}
```

## Typography

### Font Families

```css
:root {
  --font-primary: 'Inter', -apple-system, BlinkMacSystemFont,
                  'Segoe UI', 'Helvetica Neue', Arial, sans-serif;
  --font-heading: 'Georgia', 'Times New Roman', Times, serif;
  --font-mono: 'Courier New', Courier, monospace;
}
```

#### Font Usage

- **Primary (Inter)**: Body text, UI elements, navigation
- **Heading (Georgia)**: All headings (h1-h6), special text elements
- **Monospace**: Code snippets, technical content

### Font Sizes

```css
:root {
  --text-xs: 0.75rem;    /* 12px */
  --text-sm: 0.875rem;   /* 14px */
  --text-base: 1rem;     /* 16px */
  --text-lg: 1.125rem;   /* 18px */
  --text-xl: 1.25rem;    /* 20px */
  --text-2xl: 1.5rem;    /* 24px */
  --text-3xl: 1.875rem;  /* 30px */
  --text-4xl: 2.25rem;   /* 36px */
  --text-5xl: 3rem;      /* 48px */
}
```

### Font Weights

```css
:root {
  --font-normal: 400;
  --font-medium: 500;
  --font-semibold: 600;
  --font-bold: 700;
}
```

### Line Heights

```css
:root {
  --leading-tight: 1.25;
  --leading-normal: 1.5;
  --leading-relaxed: 1.75;
}
```

## Spacing System

The spacing system is based on a 4px grid for consistency.

```css
:root {
  --spacing-1: 0.25rem;   /* 4px */
  --spacing-2: 0.5rem;    /* 8px */
  --spacing-3: 0.75rem;   /* 12px */
  --spacing-4: 1rem;      /* 16px */
  --spacing-5: 1.25rem;   /* 20px */
  --spacing-6: 1.5rem;    /* 24px */
  --spacing-8: 2rem;      /* 32px */
  --spacing-10: 2.5rem;   /* 40px */
  --spacing-12: 3rem;     /* 48px */
  --spacing-16: 4rem;     /* 64px */
  --spacing-20: 5rem;     /* 80px */
  --spacing-24: 6rem;     /* 96px */
}
```

### Spacing Usage

- **Small gaps**: `--spacing-2` to `--spacing-4` (8-16px)
- **Medium gaps**: `--spacing-6` to `--spacing-8` (24-32px)
- **Large gaps**: `--spacing-12` to `--spacing-16` (48-64px)
- **Section spacing**: `--spacing-20` to `--spacing-24` (80-96px)

## Breakpoints

Responsive breakpoints for different screen sizes.

```css
:root {
  --breakpoint-sm: 640px;   /* Mobile landscape */
  --breakpoint-md: 768px;   /* Tablet portrait */
  --breakpoint-lg: 1024px;  /* Tablet landscape / Desktop */
  --breakpoint-xl: 1280px;  /* Large desktop */
}
```

### Media Query Usage

```css
/* Mobile-first approach */
.component {
  /* Mobile styles (default) */
  padding: var(--spacing-4);
}

@media (min-width: 768px) {
  /* Tablet and up */
  .component {
    padding: var(--spacing-8);
  }
}

@media (min-width: 1024px) {
  /* Desktop and up */
  .component {
    padding: var(--spacing-12);
  }
}
```

## Shadows

Subtle shadows for depth and elevation.

```css
:root {
  --shadow-sm: 0 1px 2px 0 rgba(0, 0, 0, 0.05);
  --shadow-md: 0 4px 6px -1px rgba(0, 0, 0, 0.1);
  --shadow-lg: 0 10px 15px -3px rgba(0, 0, 0, 0.1);
  --shadow-xl: 0 20px 25px -5px rgba(0, 0, 0, 0.1);
}
```

### Shadow Usage

- **Cards**: `--shadow-md`
- **Modals**: `--shadow-lg`
- **Dropdowns**: `--shadow-xl`
- **Buttons (hover)**: `--shadow-sm`

## Border Radius

Consistent border radius values.

```css
:root {
  --radius-sm: 0.25rem;   /* 4px */
  --radius-md: 0.5rem;    /* 8px */
  --radius-lg: 0.75rem;   /* 12px */
  --radius-xl: 1rem;      /* 16px */
  --radius-full: 9999px;  /* Fully rounded */
}
```

## Transitions

Smooth animations and transitions.

```css
:root {
  --transition-fast: 150ms ease-in-out;
  --transition-base: 250ms ease-in-out;
  --transition-slow: 350ms ease-in-out;
}
```

### Transition Usage

```css
.button {
  background-color: var(--color-primary);
  transition: background-color var(--transition-base);
}

.button:hover {
  background-color: var(--color-accent);
}
```

## Component Examples

### Button Styles

```css
.btn {
  padding: var(--spacing-3) var(--spacing-6);
  font-family: var(--font-primary);
  font-weight: var(--font-semibold);
  border-radius: var(--radius-md);
  transition: all var(--transition-base);
}

.btn-primary {
  background-color: var(--color-primary);
  color: white;
}

.btn-primary:hover {
  background-color: var(--color-accent);
  box-shadow: var(--shadow-md);
}
```

### Card Styles

```css
.card {
  background-color: var(--color-bg);
  border: 1px solid var(--color-border);
  border-radius: var(--radius-lg);
  padding: var(--spacing-6);
  box-shadow: var(--shadow-sm);
  transition: box-shadow var(--transition-base);
}

.card:hover {
  box-shadow: var(--shadow-md);
}
```

### Heading Styles

```css
h1, h2, h3, h4, h5, h6 {

  line-height: var(--leading-tight);
  color: var(--color-text);
}

  margin-bottom: var(--spacing-8);
}

h2 {
  font-size: var(--text-4xl);
  margin-bottom: var(--spacing-6);
}

h3 {
  font-size: var(--text-3xl);
  margin-bottom: var(--spacing-4);
}
```

## Accessibility

### Focus States

```css
:focus-visible {
  outline: 2px solid var(--color-primary);
  outline-offset: 2px;
}
```

### Color Contrast

All color combinations meet WCAG 2.1 AA standards:
- Body text: 4.5:1 minimum contrast ratio
- Large text (18pt+): 3:1 minimum contrast ratio
- UI components: 3:1 minimum contrast ratio

## Design Tokens Location

All design tokens are defined in:

```
site/src/styles/globals.css
```

This file is imported in `main.js` and available globally throughout the application.
