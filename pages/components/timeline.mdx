# Credentials Timeline Component

A visual timeline component for displaying chronological credentials, qualifications, and career milestones.

## Location

`site/src/components/Common/Timeline.vue`

## Features

- Chronological display of events
- Visual timeline with connecting line
- Icon support for each event
- Date and description display
- Responsive layout
- Alternating left/right layout (desktop)
- Vertical stacked layout (mobile)

## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `items` | `Array<TimelineItem>` | Yes | `[]` | Array of timeline events |
| `alternating` | `Boolean` | No | `true` | Alternate items left/right on desktop |

### TimelineItem Structure

```typescript
interface TimelineItem {
  id: string | number
  date: string           // e.g., "2018" or "Januar 2020"
  title: string          // e.g., "Approbation"
  description: string    // Event description
  icon?: string          // Optional icon URL
  type?: string          // Optional: 'education', 'certification', 'experience'
}
```

## Component Code

```vue
<script setup>
const props = defineProps({
  items: {
    type: Array,
    required: true,
    default: () => []
  },
  alternating: {
    type: Boolean,
    default: true
  }
})

const getItemClass = (index) => {
  if (!props.alternating) return ''
  return index % 2 === 0 ? 'timeline-item-left' : 'timeline-item-right'
}
</script>

<template>
  <div class="timeline">
    <div class="timeline-line"></div>

    <div
      v-for="(item, index) in items"
      :key="item.id"
      class="timeline-item"
      :class="[getItemClass(index), `timeline-type-${item.type || 'default'}`]"
    >
      <div class="timeline-marker">
        <div v-if="item.icon" class="timeline-icon">
          <img :src="item.icon" :alt="item.title" />
        </div>
        <div v-else class="timeline-dot"></div>
      </div>

      <div class="timeline-content">
        <div class="timeline-date">{{ item.date }}</div>
        <h3 class="timeline-title">{{ item.title }}</h3>
        <p class="timeline-description">{{ item.description }}</p>
      </div>
    </div>
  </div>
</template>
```

## Best Practices

1. **Chronological Order**: Display items in reverse chronological order (newest first)
2. **Consistent Dates**: Use consistent date format throughout
3. **Concise Content**: Keep titles short, descriptions 1-2 sentences
4. **Visual Hierarchy**: Use typography to establish clear hierarchy
5. **Mobile-First**: Design for mobile, enhance for desktop

## Testing

```javascript
import { mount } from '@vue/test-utils'
import Timeline from '@/components/Common/Timeline.vue'

describe('Timeline', () => {
  const items = [
    { id: 1, date: '2020', title: 'Title 1', description: 'Desc 1' },
    { id: 2, date: '2019', title: 'Title 2', description: 'Desc 2' }
  ]

  it('renders all timeline items', () => {
    const wrapper = mount(Timeline, {
      props: { items }
    })

    expect(wrapper.findAll('.timeline-item')).toHaveLength(2)
  })

  it('displays dates and titles correctly', () => {
    const wrapper = mount(Timeline, {
      props: { items }
    })

    expect(wrapper.text()).toContain('2020')
    expect(wrapper.text()).toContain('Title 1')
  })
})
```
