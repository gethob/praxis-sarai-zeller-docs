# Footer Component

The Footer component provides contact information, quick links, and legal information at the bottom of every page.

## Location

`site/src/components/Navigation/Footer.vue`

## Features

- Practice contact information
- Quick navigation links
- Social media links (if applicable)
- Legal links (Impressum, Datenschutz)
- Responsive multi-column layout
- Copyright information

## Props

This component doesn't accept any props - it displays static footer content.

## Usage

```vue
<template>
  <div id="app">
    <Header />
    <router-view />
    <Footer />
  </div>
</template>

<script setup>
import Header from '@/components/Navigation/Header.vue'
import Footer from '@/components/Navigation/Footer.vue'
</script>
```

## Responsive Behavior

### Desktop (≥768px)
- Three-column grid layout
- Equal width columns
- Horizontal spacing between sections

### Tablet (640px - 767px)
- Two-column grid layout
- Stacked layout for smaller sections

### Mobile (<640px)
- Single column layout
- Full-width sections
- Increased vertical spacing

```css
@media (max-width: 767px) {
  .footer-content {
    grid-template-columns: 1fr;
    gap: var(--spacing-12);
  }
}
```

## Customization

### Adding Social Media Links

```vue
<script setup>
const socialLinks = [
  { name: 'Facebook', url: 'https://facebook.com/...', icon: 'facebook' },
  { name: 'Instagram', url: 'https://instagram.com/...', icon: 'instagram' }
]
</script>

<template>
  <div class="footer-section">
    <h3>Folgen Sie uns</h3>
    <div class="social-links">
      <a
        v-for="social in socialLinks"
        :key="social.name"
        :href="social.url"
        target="_blank"
        rel="noopener noreferrer"
        :aria-label="`Visit our ${social.name} page`"
      >
        {{ social.name }}
      </a>
    </div>
  </div>
</template>
```

## Best Practices

1. **Keep It Simple**: Footer should be informative but not overwhelming
2. **Mobile-Friendly**: Ensure all links are easily tappable on mobile
3. **Updated Copyright**: Use `currentYear` to keep copyright date current
4. **Legal Compliance**: Always include Impressum and Datenschutz for German websites
5. **Contact Access**: Make contact information easily accessible
